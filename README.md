# 运动会加油稿生成器

## 介绍

**本生成器仅供测试中文UI渲染等使用，请勿将生成的文稿在任何正式场合使用，一切造成的结果由使用者自负，开发者概不负责！**

这是一个运动会加油稿生成器，可以指定人物、班级、项目、字数等参数，随机生成一份加油稿。

## 使用

### 在线体验

#### GitHub Pages

https://nonormalcreeper.github.io/add-bullshit-oil/

### 本地使用

克隆本仓库到本地，然后在本地打开 `index.html` 即可。
